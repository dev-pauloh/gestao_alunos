# Gestão de Alunos
## Descrição
Este é um projeto de processo seletivo que lista alunos que estão
com matriculas ativas ou inativas.
## Funcionalidades
- CRUD de alunos com checkbox indicando se estão ativos
- Projeto todo em php puro
- Arquitetura MVC
